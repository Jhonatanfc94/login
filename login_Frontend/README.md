# Frontend - Mockup de Sistema de Inventario

Este es el frontend de la aplicación, desarrollado en **Angular**, que incluye un flujo completo de autenticación (Login, Registro y Confirmación) con un diseño moderno y responsivo.

## 🚀 Tecnologías utilizadas
* **Angular** (Framework principal)
* **TypeScript**
* **HTML5 & CSS3** (Diseño tipo tarjeta con flexbox)
* **Reactive Forms** (Manejo de estados y validaciones de formularios)

## ⚙️ Requisitos previos
Asegúrate de tener instalado en tu máquina:
* [Node.js](https://nodejs.org/) (incluye npm)
* [Angular CLI](https://angular.io/cli) (`npm install -g @angular/cli`)

## 🛠️ Instalación y ejecución

1. **Clona o descarga** este repositorio y navega hasta la carpeta del frontend:
   ```bash
   cd login_Frontend

2. **Instala las dependencias del proyecto:**

Bash
npm install

2. **Inicia el servidor de desarrollo:**

Bash
ng serve -o
La aplicación se abrirá automáticamente en tu navegador en http://localhost:4200/.
